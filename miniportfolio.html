Challenges

Challenge 1: Tailwind Importing Issues

Challenge 2: Tailwind.config.js

The first major issue I ran into was improper importing of Tailwind. I was using all the correct commands, but for some reason Node Modules weren't being created. I later found out that the reason for this was because it was installing, not in my project folder, but in my User folder on my computer. After dragging that file into my project, most issues fixed itself.


While everything was working overall, some of the styles weren't showing up. I even tried downloading Tailwind IntelliSense to make sure I was inputting the classes correctly. I found that they weren't even being recognized. For about 3 hours, I researched troubleshooting for this issue. I asked Chat GPT, looked on Reddit, Stack Overflow, read the Tailwind docs over and over and over - yet nothing I tried was working. I eventually found someone was having the same issue in the Tailwind community forums. They said to make sure that I was using 'extend' inside of my theme configuration. In hindsight that seems really obvious, but I couldn't find good documentation for that in the docs. Some places it looked optional, and I couldn't find an explaination of what it was even for. I have since figured out that my custom styles were overwriting the default styles. By adding it into the 'extend', it allowed both styles to exist.
